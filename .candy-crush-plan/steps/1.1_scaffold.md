# Step 1.1: Project Scaffold

## Goal
Create the basic project structure for candy-crush library.

## What To Do

1. Create directory structure:
```
candy-crush/
├── src/
│   ├── App/
│   ├── Tui/
│   │   └── Components/
│   ├── Providers/
│   ├── Skills/
│   │   └── BuiltIn/
│   ├── Agents/
│   │   └── Types/
│   ├── Hooks/
│   ├── MCP/
│   │   └── Tools/
│   ├── Messages/
│   ├── Tools/
│   │   └── BuiltIn/
│   ├── Config/
│   ├── CLI/
│   └── Util/
├── tests/
├── examples/
├── bin/
└── .skills/
```

2. Create `composer.json`:
```json
{
    "name": "sugarcraft/candy-crush",
    "description": "TUI AI coding assistant - multi-provider, multi-agent, skill-aware",
    "type": "library",
    "license": "MIT",
    "minimum-stability": "dev",
    "prefer-stable": true,
    "require": {
        "php": "^8.3",
        "openai/openai": "^1.57",
        "guzzlehttp/guzzle": "^7.9",
        " SugarCraft/candy-core": "*",
        " SugarCraft/candy-sprinkles": "*",
        " SugarCraft/candy-forms": "*",
        " SugarCraft/candy-table": "*",
        " SugarCraft/candy-toast": "*",
        " SugarCraft/candy-pty": "*"
    },
    "require-dev": {
        "phpunit/phpunit": "^10.5"
    },
    "autoload": {
        "psr-4": {
            "SugarCraft\\Crush\\": "src/"
        }
    },
    "autoload-dev": {
        "psr-4": {
            "SugarCraft\\Crush\\Tests\\": "tests/"
        }
    },
    "repositories": [
        {
            "type": "path",
            "url": "../candy-core",
            "options": {"symlink": true}
        },
        {
            "type": "path",
            "url": "../candy-sprinkles",
            "options": {"symlink": true}
        },
        {
            "type": "path",
            "url": "../candy-forms",
            "options": {"symlink": true}
        },
        {
            "type": "path",
            "url": "../candy-table",
            "options": {"symlink": true}
        },
        {
            "type": "path",
            "url": "../candy-toast",
            "options": {"symlink": true}
        },
        {
            "type": "path",
            "url": "../candy-pty",
            "options": {"symlink": true}
        }
    ]
}
```

3. Create `phpunit.xml`:
```xml
<?xml version="1.0" encoding="UTF-8"?>
<phpunit xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
         xsi:noNamespaceSchemaLocation="vendor/phpunit/phpunit/phpunit.xsd"
         bootstrap="vendor/autoload.php"
         colors="true"
         failOnWarning="true"
         cacheDirectory=".phpunit.cache">
    <testsuites>
        <testsuite name="default">
            <directory>tests</directory>
        </testsuite>
    </testsuites>
    <coverage>
        <include>
            <directory suffix=".php">src</directory>
        </include>
    </coverage>
</phpunit>
```

4. Create basic placeholder classes:
   - `src/App/App.php` - stub
   - `src/Tui/Pane.php` - enum stub
   - `src/Providers/ProviderInterface.php` - interface stub

5. Add to root `composer.json` and `MATCHUPS.md` (follow AGENTS.md checklist)

## Files To Create
- `candy-crush/composer.json`
- `candy-crush/phpunit.xml`
- `candy-crush/README.md` (basic stub)
- `candy-crush/CALIBER_LEARNINGS.md`
- `candy-crush/src/App/App.php`
- `candy-crush/src/Tui/Pane.php`
- `candy-crush/src/Providers/ProviderInterface.php`

## Verification
- Run `composer validate` (will flag sugarcraft/* as expected, ignore)
- Run `vendor/bin/phpunit` - should pass (or show no tests yet)

## Notes
- Follow PSR-12 strictly
- Use `declare(strict_types=1);` in all files
- Public classes should be `final` unless meant for extension
