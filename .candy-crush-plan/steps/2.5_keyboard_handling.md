# Step 2.5: Keyboard Handling

## Goal
Implement keyboard event handling and pane navigation.

## What To Do

Create `src/Tui/KeyboardHandler.php`:

```php
declare(strict_types=1);

namespace SugarCraft\Crush\Tui;

use SugarCraft\Crush\App\App;
use SugarCraft\Crush\Tui\Components\MenuBar;

/**
 * Handles keyboard input and routes to appropriate handlers.
 */
final class KeyboardHandler
{
    /**
     * Process a keypress and return updated App and optional command.
     *
     * @return array{App, ?object} [newApp, command]
     */
    public function handle(string $key, App $app): array
    {
        // Handle Tab - cycle panes
        if ($key === 'tab') {
            return [$app->withPane($app->pane->next()), null];
        }

        // Handle arrow keys / vim keys for navigation
        if (in_array($key, ['up', 'k', 'down', 'j', 'left', 'h', 'right', 'l'])) {
            return $this->handleNavigation($key, $app);
        }

        // Handle menu shortcuts
        if (MenuBar::getMenuItems($app->pane->label()) !== []) {
            $result = MenuBar::handleKey($key, $app);
            if ($result[1] !== null) {
                return [$app, $result[1]];
            }
        }

        // Handle Escape
        if ($key === 'escape') {
            MenuBar::closeMenu();
            return [$app->withPane(Pane::Chat), null];
        }

        // Handle Ctrl+key combinations
        if (str_starts_with($key, 'ctrl+')) {
            return $this->handleCtrl(substr($key, 5), $app);
        }

        return [$app, null];
    }

    private function handleNavigation(string $key, App $app): array
    {
        // Navigation within panes would be handled by specific pane handlers
        // For now, just pass through
        return [$app, null];
    }

    private function handleCtrl(string $key, App $app): array
    {
        return match ($key) {
            'n' => [$app, new NewSessionCmd()],
            'c' => [$app, new CancelCmd()],
            'g' => [$app, new GroupInputCmd()],
            'k' => [$app, new CommandPaletteCmd()],
            's' => [$app, new SourceSkillCmd()],
            'a' => [$app->withPane(Pane::Agents), null],
            'p' => [$app, new ProviderSelectCmd()],
            ',' => [$app->withPane(Pane::Settings), null],
            default => [$app, null],
        };
    }
}

// Keyboard command types
interface KeyCmd {}

final readonly class NewSessionCmd implements KeyCmd {}
final readonly class CancelCmd implements KeyCmd {}
final readonly class GroupInputCmd implements KeyCmd {}
final readonly class CommandPaletteCmd implements KeyCmd {}
final readonly class SourceSkillCmd implements KeyCmd {}
final readonly class ProviderSelectCmd implements KeyCmd {}
```

## Keyboard Reference (for documentation)
| Key | Action |
|-----|--------|
| Tab | Cycle through panes |
| ↑/↓ or j/k | Navigate lists |
| left/right or h/l | Navigate menus |
| Enter | Select / Send |
| Escape | Cancel / Back |
| Ctrl+N | New session |
| Ctrl+C | Cancel operation |
| Ctrl+G | Multi-line input |
| Ctrl+K | Command palette |
| Ctrl+S | Apply skill |
| Ctrl+A | Agent panel |
| Ctrl+P | Provider selector |
| Ctrl+, | Settings |
| q | Quit |
