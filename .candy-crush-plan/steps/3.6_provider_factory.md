# Step 3.6: Provider Factory

## Goal
Create a factory for creating providers from configuration.

## What To Do

Create `src/Providers/ProviderFactory.php` - a factory class that creates providers from config arrays with environment variable resolution support.

Key methods:
- `create(array|string $config): ProviderInterface` - main factory method
- `resolveEnv(?string $value): ?string` - resolves ${VAR} and ${VAR:-default} patterns
- `availableTypes(): array` - list of supported provider types
- `defaultConfig(string $type): array` - get default config for a type

Supported types: openai, anthropic, claude-code, sglang, bedrock, vertex, custom

## Files To Create
- `candy-crush/src/Providers/ProviderFactory.php`

## Verification
- Creates providers from config array
- Environment variable resolution
- Default configs for each type
