# Step 1.2: Provider Interface and Message Classes

## Goal
Create the Provider interface and base message classes.

## What To Do

### Provider Interface (`src/Providers/ProviderInterface.php`)

```php
declare(strict_types=1);

namespace SugarCraft\Crush\Providers;

use SugarCraft\Crush\Messages\Message;
use SugarCraft\Crush\Tools\ToolCall;

interface ProviderInterface
{
    public function name(): string;

    public function supportsStreaming(): bool;

    public function supportsFunctionCalling(): bool;

    public function supportsVision(): bool;

    public function supports_json_schema(): bool;

    public function contextWindow(): int;

    public function costPer1kTokens(string $model, 'input'|'output'): float;

    public function complete(CompleteRequest $request): CompleteResponse;

    public function completeStream(CompleteRequest $request): \Generator;

    public function embeddings(EmbeddingsRequest $request): EmbeddingsResponse;
}

final readonly class CompleteRequest
{
    public function __construct(
        public string $model,
        public array $messages,
        public ?array $tools = null,
        public ?string $systemPrompt = null,
        public ?float $temperature = null,
        public ?int $maxTokens = null,
        public ?string $jsonSchema = null,
    ) {}
}

final readonly class CompleteResponse
{
    public function __construct(
        public string $content,
        public ?string $reasoning = null,
        public ?array $toolCalls = null,
        public int $tokensUsed = 0,
        public float $costUsd = 0.0,
    ) {}
}

final readonly class EmbeddingsRequest
{
    public function __construct(
        public string $model,
        public array $input,
    ) {}
}

final readonly class EmbeddingsResponse
{
    public function __construct(
        public array $embeddings,
    ) {}
}
```

### Message Classes

Create `src/Messages/Message.php`:
```php
declare(strict_types=1);

namespace SugarCraft\Crush\Messages;

interface Message
{
    public function role(): string;
    public function content(): string;
    public function toArray(): array;
}
```

Create `src/Messages/UserMessage.php`:
```php
declare(strict_types=1);

namespace SugarCraft\Crush\Messages;

final readonly class UserMessage implements Message
{
    public function __construct(
        private string $content,
    ) {}

    public function role(): string => 'user';
    public function content(): string => $this->content;
    public function toArray(): array => ['role' => 'user', 'content' => $this->content];
}
```

Create `src/Messages/AssistantMessage.php`:
```php
declare(strict_types=1);

namespace SugarCraft\Crush\Messages;

use SugarCraft\Crush\Tools\ToolCall;

final readonly class AssistantMessage implements Message
{
    public function __construct(
        private string $content,
        private ?array $toolCalls = null,
        private ?string $reasoning = null,
    ) {}

    public function role(): string => 'assistant';
    public function content(): string => $this->content;
    public function toolCalls(): ?array => $this->toolCalls;
    public function reasoning(): ?string => $this->reasoning;
    public function toArray(): array => [
        'role' => 'assistant',
        'content' => $this->content,
        'tool_calls' => $this->toolCalls,
        'reasoning' => $this->reasoning,
    ];
}
```

Create `src/Messages/ToolResultMessage.php`:
```php
declare(strict_types=1);

namespace SugarCraft\Crush\Messages;

final readonly class ToolResultMessage implements Message
{
    public function __construct(
        private string $toolCallId,
        private string $content,
        private bool $isError = false,
    ) {}

    public function role(): string => 'tool';
    public function content(): string => $this->content;
    public function toolCallId(): string => $this->toolCallId;
    public function isError(): bool => $this->isError;
    public function toArray(): array => [
        'role' => 'tool',
        'tool_call_id' => $this->toolCallId,
        'content' => $this->content,
    ];
}
```
